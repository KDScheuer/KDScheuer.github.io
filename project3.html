Password Manager

Introduction

I developed a password management application in Python, designed for the persistent and secure storage of passwords. The key technologies utilized in this project include:

  • AES encryption
  • SHA512
  • GUI using TKinter

Implentation

Technologies Used: Python

Upon the application's first run, it checks for the existence of a key file, which it generates for encrypting user information. The key file is itself encrypted with a master password. If the key file is present, the user is prompted to enter the password to decrypt it. Otherwise, the program guides the user in setting a master password and generates a key file for that user.

The key file is encrypted with a master password, so if the key file is present it will prompt the user for the password to decrpyt the key. Otherwise the program will prompt the user to set a master password and generate a key file for that user.

Once the key file is decrypted, the passwords become accessible, and the application displays the main menu. Users can choose to view their accounts, add a new account, or exit the application.

If the user opts to view an account, they are presented with options to delete or modify the account.

Challenges Faced

The primary challenge encountered during this project revolved around creating the GUI using TKinter, as I was initially unfamiliar with it. Additionally, I gained valuable insights into UTF-8 and the various text formats involved in encoding, decoding, encryption, and decryption processes.
