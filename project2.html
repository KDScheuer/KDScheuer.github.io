World Map API

Introduction

This is a straightforward program that enables users to hover over any country on a world map and access information such as its capital city, region, and income level. The project utilizes the World Bank API.

Implentation

Technologies Used: Angular, JavaScript, HTML, CSS

In the design of this project, the emphasis was on providing a user-friendly experience. The country being hovered over changes color immediately, reverting to black upon exit. This ensures that users are always aware of their mouse position on the screen.

When the mouse hovers over a country, its color changes, and the current mouse position is checked against the world map SVG to obtain the country's ID. Subsequently, an API call is made to the World Bank's API to retrieve the country's information. Any errors encountered during this process are logged to the console.

Upon receiving the API response, the data is parsed, and an object with the relevant attributes is created. This object is then sent to another function that renders the correct data in the aside element on the webpage.

Challenges Faced

This project was undertaken for a college course and required the use of the Angular framework, which posed a significant obstacle as I was not familiar with it. After configuring a proxy to facilitate successful API requests, the main challenge involved updating the aside element with the obtained data.
